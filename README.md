# hoshino_asgacha
hoshinobot的llas抽卡模拟器v1.1

# 使用方法
 把image.zip解压后放到\res\img\  下
 
 在__bot__.py内添加 'asgacha'
 
 稍微修改下代码中的几个路径位置，使得文件能被正确读取（我懒 没有写成相对位置
